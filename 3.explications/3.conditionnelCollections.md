# Affichage conditionnel et collections

- En React, il est courant de devoir afficher des éléments de manière conditionnelle ou de rendre des collections d'éléments basées sur des données.

## Affichage conditionnel
- Pour afficher des éléments de manière conditionnelle, on peut utiliser des expressions JavaScript à l'intérieur du JSX. Par exemple, on peut utiliser l'opérateur ternaire ou les opérateurs logiques.

```jsx
function Greeting({ isLoggedIn }) {
  return (
    <div>
      {isLoggedIn ? <h1>Bienvenue de nouveau!</h1> : <h1>Veuillez vous connecter.</h1>}
    </div>
  );
}
```

- Il est possible d'utiliser null comme retour pour ne rien afficher.

```jsx
function Greeting({ isLoggedIn }) {
  return (
    <div>
      {isLoggedIn ? <h1>Bienvenue de nouveau!</h1> : null}
    </div>
  );
}
```

- il est également possible d'utiliser l'opérateur logique `&&` pour un affichage conditionnel.

```jsx
function Greeting({ isLoggedIn }) {
  return (
    <div>
      {isLoggedIn && <h1>Bienvenue de nouveau!</h1>}
    </div>
  );
}
```

## Rendu de collections

- Pour rendre des collections d'éléments basées sur des données, on utilise généralement la méthode `map()` pour itérer sur un tableau et retourner un élément JSX pour chaque item.

```jsx
function ItemList({ items }) {
  return (
    <ul>
      {items.map(item => (
        <li key={item.id}>{item.name}</li>
      ))}
    </ul>
  );
}
```

- Il est important de fournir une prop `key` unique pour chaque élément rendu dans une liste afin d'aider React à identifier quels éléments ont changé, ont été ajoutés ou supprimés.

```jsx
<li key={item.id}>{item.name}</li>
```

- La clé doit être stable, unique et ne pas changer entre les rendus. Évitez d'utiliser des index de tableau comme clés si l'ordre des éléments peut changer.

```jsx

// Mauvaise pratique ( dernier recours )
{items.map((item, index) => (
  <li key={index}>{item.name}</li>
))}

// Bonne pratique
{items.map(item => (
  <li key={item.id}>{item.name}</li>
))}
```

- En combinant l'affichage conditionnel et le rendu de collections, on peut créer des interfaces utilisateur dynamiques et réactives en React.

```jsx
function TodoList({ todos }) {
    return (
        <ul>
        {todos.map(todo => (
            <li key={todo.id}>
            {todo.completed ? <s>{todo.text}</s> : todo.text}
            </li>
        ))}
        </ul>
    );
    }
```
