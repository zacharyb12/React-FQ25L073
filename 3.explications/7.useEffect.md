# UseEffect 

- Le hook `useEffect` permet de gérer les effets de bord dans les composants fonctionnels de React. Un effet de bord est une opération qui interagit avec le monde extérieur au composant, comme la manipulation du DOM, les appels API, les abonnements à des événements, etc.

### Syntaxe de useEffect

```jsx
useEffect( () => { } , [] )
```

on retrouve deux elements principaux :
- le code a executer 
- un tableau de dépendances

### Sans Dépendances
- Si le tableau de dépendances est vide (`[]`), l'effet ne s'exécutera qu'une seule fois, après le premier rendu du composant (comportement similaire à `componentDidMount` dans les classes).

```jsx
useEffect(() => {
  console.log('Composant monté');
}, []);
```

### Avec Dépendances
- Si le tableau de dépendances contient des variables, l'effet s'exécutera après chaque rendu où l'une des dépendances a changé.

```jsx
const [count, setCount] = useState(0);

useEffect(() => {

  console.log('Count a changé:', count);

}, [count]);
```

### Nettoyage des Effets
- Si l'effet crée des abonnements ou des timers, il est important de nettoyer ces ressources pour éviter les fuites de mémoire. Pour cela, on peut retourner une fonction de nettoyage depuis l'effet.

```jsx
useEffect(() => {

  return () => {
    // Code de nettoyage ici
    console.log('Composant démonté');
  };
}, []);
```
