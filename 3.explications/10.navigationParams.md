# Les paramètres de navigation : useParams et useNavigate

### useParams
- Le hook `useParams` est utilisé pour accéder aux paramètres dynamiques dans l'URL d'une route.
- Par exemple, si vous avez une route définie comme `/user/:id`, vous pouvez utiliser `useParams` pour récupérer la valeur de `id`.

### useNavigate
- Le hook `useNavigate` est utilisé pour naviguer entre les routes.
- Il permet d'ajouter des valeurs dans l'URL ou de rediriger l'utilisateur vers une autre page.

### Definir un paramètre dans la route
- La route doit être définie avec un paramètre dynamique en utilisant les deux-points `:` suivi du nom du paramètre. (/:id )
- Le même nom doit être utilisé pour récupérer la valeur avec `useParams`.


### Définition des routes avec paramètres
```jsx
import { BrowserRouter as Router, Routes, Route } from "react-router-dom";
import UserProfile from "./UserProfile";

function App() {
    return (
        <Router>
            <Routes>
                <Route path="/user/:id" element={<UserProfile />} />
            </Routes>
        </Router>
    );
}

export default App;
```


### Exemple d'utilisation de useNavigate
```jsx
import { useParams, useNavigate } from "react-router-dom";

function listProfile() {
    const navigate = useNavigate();

    const handleNavigate = () => {
        navigate(`/user/${id}/edit`);
    };

    return (
        <div>
            <h1>User Profile</h1>
            <p>ID: {id}</p>
            <button onClick={handleNavigate}>Profile</button>
        </div>
    );
}
export default listProfile;
```

### Exemple d'utilisation de useParams 
```jsx
import { useParams } from "react-router-dom";

function UserProfile() {
    const { id } = useParams();

    return (
        <div>
            <h1>User Profile</h1>
            <p>ID: {id}</p>
        </div>
    );
}

export default UserProfile;
```
